# GetDealScout

Automated retail arbitrage alert service - Costco to Amazon opportunities delivered daily.

## Architecture

- **Scraper:** Python + GitHub Actions
- **Database:** Airtable
- **AI:** OpenAI GPT-4o-mini (data extraction)
- **Pricing:** Keepa API
- **Email:** SendGrid
- **Payment:** Gumroad

## Stack

- Python 3.11+
- Airtable (database & templates)
- OpenAI (HTML parsing)
- Keepa (Amazon pricing)
- SendGrid (email delivery)
- Jinja2 (email templates)
